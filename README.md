# Bike Go Rush 🏍️💨

An exciting, high-speed HTML5 endless runner game built with pure web technologies. Dodge random obstacles, collect coins, and unlock powerful rides to chase the ultimate high score!

## 🚀 Features

* **7 Unique Bikes:** Unlock and ride 7 different bikes, each with its own style.
* **Coin & Shop System:** Collect coins during your run and use them to purchase upgrades or new bikes.
* **Progressive Speed Scaling:** The longer you survive, the faster and more intense the game becomes!
* **Dynamic Obstacles:** Randomly generated obstacles keep every run fresh and challenging.
* **High Score Tracker:** Saves your personal best score so you can always try to beat it.
* **Offline Play:** Fully functional offline once loaded. Perfectly optimized for both mobile and desktop browsers.

---

## 🛠️ Tech Stack Used

* **HTML5:** For core game structure and elements.
* **CSS3:** For responsive layout, styling, and animations.
* **JavaScript (ES6):** For core game loop, collision detection, score calculation, and shop mechanics.

---

## 🕹️ How to Play

1. **Start the Game:** Click the play button on the screen.
2. **Move/Jump:** Use the **Arrow Keys** (or Spacebar) on your keyboard (or tap on the screen for mobile) to dodge incoming obstacles.
3. **Collect Coins:** Grab as many shiny coins as possible along the track.
4. **Visit the Shop:** Spend your hard-earned coins to unlock cooler, faster bikes.
5. **Survive:** Don't crash! Keep going to set a new world record.
